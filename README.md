# Noxun Engine

Parametrický nábytkársky systém pre SketchUp (Ruby plugin). Korpusy generované kódom z konfigurácie — žiadne Dynamic Components: zóny s priečkami, čelá s lockmi (fixed/auto), šablóny, scale→automatická prestavba na mm.

- **Špecifikácia:** Noxun Component Standard (`SYSTEM/01_STANDARD_draft.md` v workspace)
- **Inštalácia (dev):** `INSTALL_noxun_engine.ps1` → SketchUp 2026 → Extensions → Noxun Engine → Panel
- **Workflow:** vetva → PR → Codex review → merge (viď `CLAUDE.md`)

Stav: **v0.2.1** — jadro korpusu (dolná/horná, konštrukčné varianty, strom zón, čelá, šablóny). Ďalej: materiály+ABS, kovanie, výstupy (kusovník, VEPO).
